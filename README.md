## Collab repo
Hi, team let's collob on this repo by simply adding a endpoint with your name.
Steps:
1. Clone the repo.
2. Create a branch with your name.
3. Add the 'hello from <name>' endpoint to it.
4. Create a pull request to main branch.
5. **Most Importantly: Be aware of merge confilcts never override other members endpoint with yours, if conflict arise take the latest pull from main keep both changes, validate locally and push please. 🙏**
5. Validate your changes [here](https://collab-flask-hello-783982891559.us-central1.run.app/).
